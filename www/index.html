Кого мы ищем

CTO

Для разработки и реализации стратегии развития

Обязанности

  • Блокчейн разработка
  • Подбор и формирование команды
  • Разработка бизнес требований

Требования

  • Опыт работы на руководящей позиции
  • Практический опыт работы с технологиями и стандартами NodeJS
  • Владение английским языком

System architect

Для разработки архитектуры всего продукта

Обязанности

  • Формирование и развитие технологического стека / ядра продукта
  • Участие во внедрении, тестировании и сопровождении программных продуктов
  • Написание спецификаций и WP

Требования

  • Научный опыт и фундаментальное образование в математике, компьютерных сетях, информационной безопасности или любой другой смежной области
  • Практическое знакомство с такими проектами, как Ethereum/ Stellar/ Ripple/ Hyperledger
  • Глубокое понимание криптографии, включая асимметричные (pub / priv key), симметричные хэш-функции, шифрование / подписи

Team lead

Для руководства процессом разработки проекта

Обязанности

  • Создание выдающейся Front-end команды. Контролирование и распределение задач
  • Взаимодействие с внешними подрядчиками, внутренними заказчиками и другими командами
  • Спроектировать архитектуру, сформулировать стратегию развития пользовательского кода в продукте

Требования

  • Навыки по работе с системой контроля версии Git, CI/CD
  • Хорошее понимание JavaScript и свободное владение NodeJS
  • Умение производить экспертизу в области разработки client side. Возможность работать со старым кодом и приводить его в порядок

Senior C++ Developer

Для разработки ПО на языке C++ под Linux

Обязанности

  • Доработка различных частей существующего кода в соответствии с требованиями
  • Участие в проектировании архитектуры ПО
  • Исправление ошибок, поддержка существующего кода

Требования

  • Отличное знание языка C++
  • Опыт разработки многопоточных приложений
  • Владение основными средствами разработки и контроля версий: Visual Studio, git и т.д

DevOps

Для внедрения и эксплуатации продукта

Обязанности

  • Автоматизация процессов непрерывной интеграции продукции и обеспечение бесперебойной работы
  • Развитие и поддержка системы мониторинга инфраструктуры, анализ нагрузки и прогнозирование потребностей в расширении
  • Проведение исследовательской работы и тестирования аппаратных платформ и программных компонентов, поиск и устранение узких мест в архитектуре системы, оптимизация производительности

Требования

  • Знания стека TCP/IP
  • Администрирование и поддержка серверной инфраструктуры на базе Linux
  • Мониторинг на базе Grafhana, Zabbix, InfluxDB
  • Умение работать с Docker, Kubernetes
  • Опыт работы с облачными платформами: AWS, Heroku, Google Cloud

Javascript front-end Developer

Для разработки и реализации нового функционала продукта

Требования

  • Уверенное знание JavaScript (vanilla js)
  • Знание HTML5, CSS3, JQuery, SASS/LESS
  • Опыт работы с Electron / Vue.js / React / Angular 2+, TypeScript
  • Умение работать с git и понимание git flow
  • Опыт написания unit-тестов
  • Базовые знания в современной криптографии: ассиметричное шифрование, hash, цифровая подпись
  • Знание принципов работы блокчейн систем, в т.ч. смарт контрактов
  • Знание принципов и опыт разработки API (REST, JSON-RPC)

Желательно

  • Опыт работы с Ethereum (Solidity)/ Hyperledger Fabric
  • Знание сетевых протоколов, криптографии. Знание Python
